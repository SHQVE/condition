# 🍰 Delice — сайт кондитерской

Веб-сайт кондитерской **Delice**, разработанный для демонстрации ассортимента десертов и оформления заказов.

---

## О проекте

Проект представляет собой многостраничный сайт с основными разделами:

* Главная страница
* Меню десертов
* Информация о кондитерской
* Форма оформления заказа


### Главная страница

* Навигация по сайту
* Переход к меню

### Меню

* Категории:

  * Торты
  * Конфеты
  * Авторские десерты
* Карточки товаров с описанием и ценой

### Оформление заказа

* Модальное окно (Bootstrap)
* Автоматическая подстановка выбранного товара
* Проверка заполнения формы
* Уведомление об успешном заказе

### Викторина

* Помогает выбрать десерт
* Показывает результат на основе ответа

### Адаптивность

* Поддержка мобильных устройств
* Перестройка интерфейса под разные экраны
